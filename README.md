# Cloudflare D1 Try

## Getting Started

1. Install the packages.

```bash
npm Install
```

2. Rust download.

https://www.rust-lang.org/tools/install

3. Run the server on local.

```bash
npm run dev
```

5. Run the build.

```bash
npm run build
```

6. Run the publish.

```bash
npm run login
npm run publish
```
